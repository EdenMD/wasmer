How To Use Editor

Gen1 Project Editor Guide

File Explorer (Drawer)

Access and manage your project files using the file explorer.

  • Tap the Drawer button in the top-right corner to open/close the file explorer.
  • Creating Files/Folders: Use the New File and New Folder buttons inside the drawer to add new entries.
  • Uploading Files: Use the Upload File button to import files from your device.
  • Opening Files: Click on any file in the list to open it in the editor.
  • Renaming: Hover over a file or folder and click the icon to rename it.
  • Deleting: Hover over a file or folder and click the icon to delete it. You'll be asked for confirmation.
  • Folders: Click on folder names to expand or collapse them.

Code Editor

The main area where you write and modify your code.

  • Typing: Simply type your code directly into the editor.
  • Syntax Highlighting: The editor automatically provides syntax highlighting for many common languages (HTML, CSS, JavaScript, Python, etc.).
  • Automatic Layout: The editor automatically adjusts its size to fit the available space.

Bottom Navigation Bar

Quick actions and tools are available at the bottom of the screen.

  • AI Assist: (Future Feature) This button is reserved for AI-powered coding assistance.
  • Format Code: Automatically formats the code in the current open file to improve readability and consistency.
  • Save File: Manually saves the changes you've made to the currently open file into the project database.
  • Copy Code: Copies the entire content of the current file to your clipboard.
  • How To Use: You're currently here! Access this guide for help.

Live Preview (Run Button)

For HTML files, you can quickly preview your work.

  • When an HTML file is open, the Run button will appear in the top-right bar.
  • Click it to open a live preview of your HTML file in a new browser tab.
  • If the preview doesn't open, ensure your browser allows pop-ups.

/ Theme Toggle

Switch between light and dark modes.

  • Use the / button in the top-right corner to toggle between light and dark themes for the entire application.

Back Button

Return to the main project list.

  • Click the button in the top-left corner to go back to the Welcome page and manage your projects.
